== Description ==

Image Caption Hover Gallery is a <strong>Multi Purpose</strong> Gallery to displays Images and Captions with 70+ CSS3 Animations. Responsive Grid for Images along with Pop Up Support. It Includes following incredible features.
<h3>Features</h3>
<ul>
<li>70+ Modern Hover Animations</li>
<li>Responsive Grid (1 to 12 Columns)</li>
<li>Responsive Light Box</li>
<li>Drag Drop to Sort Columns</li>
<li>Import and Export Capability</li>
<li>Gallery Categories Support</li>
<li>Dummy Content Included</li>
</ul>

== Installation ==

1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Image Caption Hover Gallery' and install it
3. Go to IH Gallery > Add New, and insert columns
4. Upload images and fill some additional informations. (Not Necessary)
5. Now paste shortcode of that gallery and Enjoy...
